    Mas só para fixar: nesse modo que estou praticando precisa colocar o comando git remote add com o para onde você vai enviar e como, toda vez tem a necessida de colocar o comando git add (nome do repositório) (e como vai ser enviado) por conta de já existir um documento assim no servidor, então é só fazer as aleterações no "editor de código" e fazer um commit para os dados serem salvos e preparados para envio, depois disso mandar com o comando "git push" Ordem de execução-------------------------------------------------- 1º Alterar documento "Editor de texto" 2º Dicionar essas alterações no git (git add . - para adicionar todos os arquivos novos e atualizados) 3º Confirmar alteração através do (git status) 4º Adicionar as novas alterações em um pacote para ser enviado para o servidor (git remote add local master) 5º Fazer o commit fazer uma descrição das atualizações, fechar o pacote (git commit -m "...") 6º Enviar para o servidor (git push local master) -------------------------------------------------------------------- 7º Depois só entrar na pasta do novo usuário e puxa as informações atualizadas (git pull local master)
